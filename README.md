# My Reads
This is a simple App that helps you categorize books you have read, are currently reading, or want to read. It also allows the user to search for books and add them to one of the shelves.

## Instructions
In order to start this app simply run `npm install` and `npm update`. After that run `npm start` to start the app.

## Packages
react-router-dom: https://github.com/ReactTraining/react-router

react-prop-types: https://github.com/react-bootstrap/react-prop-types

immutability-helper: https://github.com/kolodny/immutability-helper
